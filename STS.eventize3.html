This eventization is to categorize the original data to different # levels of event. # Description of variables: evtDef--event definition, takes either percentile or absolute value for threshold with the default percentile 0.95. # event: takes either "greater" or "lower" depending on how to define the event greater or lower than the threshold. # Modified date: May, 2019 STS.eventize3 <- function(data,evtDef="percentile",magn=2,thres=c(0.85,0.95),event="greater"){ # ----------------------------------------------- Test for errors if input data is not matrix format ------------------------------ # if(!is.matrix(data)){ print("!--------------------- ERROR #001 -----------------!") print("!
